Instructions

This is based on code written by Daniel Shiffman
https://editor.p5js.org/codingtrain/sketches/QmTx-Y_UP

To learn more about L-systems, also known as Lindenmayer Systems, please see:
https://paulbourke.net/fractals/lsys/

L-system operators used here:

Operator Meaning
F Move forward drawing a line
T Move forward without drawing a line
+ Rotate right
- Rotate left
/ Rotate right 90 degrees
\ Rotate left 90 degrees
[ Push
] Pop

Click the 'generate' button to grow the L-system

Change the branch length by using the 'Length' slider.

Change the angle by using the 'Angle' slider.

Change the stroke weight by using the 'Stroke weight' slider.

Key 'l': Save a low-resolution PNG of the current image.

Key 'e': Save a high-resolution PNG of the current image.
